# Blog App 

This project made by me individually in 5 days.
Functionalities given are: 
- Login 
- Signup
- Create blog
- Get all blogs
- Search blogs
- Mail to website owner.

## Demo-link : https://blog-application-lac.vercel.app/

## Tech stacks used :
 - NextJS
 - MongoDB
 - Redux
 - Redux-thunk
 - React-hooks
 - Chakra-ui (for styling and responsiveness)
 - Jsonwebtoken
